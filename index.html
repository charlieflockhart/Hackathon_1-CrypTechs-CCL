Game Rules
Objective:

Guess the 5-letter mystery word within 6 attempts.

Gameplay:
  • Start a New Game: Click the "New Game" button to begin.
  • Input Guess: Type your 5-letter guess into the input box and press "Submit."
  • Feedback: After each guess, the game provides feedback:
    • Green: Correct letter in the correct position.
    • Yellow: Correct letter in the wrong position.
    • Gray: Incorrect letter.
  • Attempts: You have 6 attempts to guess the correct word.
Themes:

Choose different themes like Christmas, food, tech, etc., for a variety of word lists.

Difficulty Levels:

Adjust the difficulty settings to make the game more challenging.
